# Xg_predictor  
Приложение на Kotlin для предсказания исходов футбольных матчей методами машинного обучения, находится в папке client.  
Сервер на Python, предназначенный к деплою на Heroku, находится в папке server. Вычисления для прогнозирования исходов происходят на стороне сервера.  
Ссылка на .apk файл для установки приложения https://yadi.sk/d/owZl8VtjTlW2HA
